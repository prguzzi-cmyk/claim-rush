info How the UPA → ACI Funnel Works

  1. Immediate SMS — Sent when a new lead enters the UPA outreach queue
  2. 15-min follow-up — If no reply, a follow-up SMS is sent automatically
  3. Next-morning email — If still no reply, a resource email goes out the next morning
  4. Reply detection — If the reply contains yes, help, claim, question → contact status = aci_ready
  5. Transition message — The lead is notified they're being connected to ACI
  6. ACI handoff — ACI team receives the lead with full context

If a reply contains STOP / UNSUBSCRIBE / REMOVE / QUIT, the contact is automatically opted out and suppressed from all future campaigns.
